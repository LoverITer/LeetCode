### 力扣算法总结出的工具类


|  类/包 |  用途 |
| :------------: | :------------: |
|  ArrayUtils                                                    |  处理数组的有关算法                                       |
| BinTreeUtils  |  处理二叉树有关的算法 |
|  LinkedListUtils | 处理链表的有关算法  |
| MathUtils  |  数学方法有关的算法 |
| SearchAlgorithm  |  查找算法 |
| sort包  |  排序算法的Java实现 |
|  StringUtils |  字符串处理的有关算法 |
